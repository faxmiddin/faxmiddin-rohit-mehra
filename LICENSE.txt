PROPRIETARY NOTICE — FAXWEB HORIZONTAL SCALABILITY SYSTEM

The following components are proprietary innovations of Rohit Mehra and are
protected under copyright law:

1. FAXWEB HORIZONTAL SCALABILITY ENGINE
   A novel 6-node ring consensus architecture applying FNV-1a hash-based
   SPV verification across multiple cryptocurrency exchanges simultaneously,
   combined with asynchronous Byzantine Fault Tolerance (aBFT) majority
   voting for signal validation.

2. CAUSAL SIGNAL DEPENDENCY CHAIN (CSDC)
   A directed acyclic graph-based signal dependency model mapping:
   Funding Rate → Open Interest → CVD → Bid Wall → Shannon Entropy
   → VWAP Deviation → Regime → Pump Signal
   with geometric mean chain strength and weak-link penalty.

3. DUAL-SET BYZANTINE CONSENSUS (DS-BFT)
   A Sumeragi-inspired consensus model dividing validators into
   Set A (2f+1 active) and Set B (f passive), with deterministic
   leader selection based on block hash modulo.

4. ADAPTIVE SIGMOID SIGNAL SCORING (ASSS)
   A neural-inspired weighted scoring system using sigmoid activation
   on historical accuracy metrics for each signal component.

5. QUAD CONSENSUS BONUS SYSTEM
   A four-layer validation framework combining CSDC + FAXWEB aBFT +
   DS-BFT + ASSS with additive bonus scoring (+2 to +14 points).
